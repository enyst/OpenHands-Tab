# OpenHands-Tab README Summary

1. **VS Code Extension for AI Agents** – OpenHands-Tab lets you chat with OpenHands AI agents directly in your IDE, supporting both local mode (agent runs in VS Code) and remote mode (connects to an agent-server).

2. **Rich Feature Set** – Includes streaming chat, action confirmation with security risk indicators, conversation persistence, workspace file context, and integrated terminal output.

3. **Monorepo Architecture** – The project is an npm workspace with two packages: the VS Code extension itself and `@openhands/agent-sdk-ts`, a TypeScript SDK providing conversation APIs, multiple LLM clients (Anthropic, OpenAI, Gemini), and various tools (Terminal, FileEditor, Browser, etc.).
